## Executar o game

```bash
    # Clone o Repositório:
    $ git clone https://github.com/SilvaneiMartins/flappy-bird-react-native

    # Entre no Diretório:
    cd flappy-bird-react-native


    # Instalar as dependências
    $ yarn
    $ npm install

    # Executar expo go
    $ expo start

    # Executar no device
    $ npx expo run:ios
    $ npx expo run:android
```
